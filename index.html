HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shoen to the user
  • "Li is the listed items and ul is the container for the listed items"
  • "The only visible elements are those in the body things put in the head are meta data or setup for the page"

CSS

  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element

Git

  • Git status is to tell you what branch you are currently in
  • Git add is to stage the changes that you have made so you can commit then push
  • Git pull will make sure you are up to current version on desired branch
  • Workflow is pull(from origin main), add(-A), commit(needs a message with -m and""), push(if working in branch then add origin)
  • git checkout -b creates a new branch and moves to it

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Functions are a set of instructions for the computer to execute and makes it easy to call back to the set of code later on
  • Functions need to be called after being defined or they wont be ran
  • Arays are a container of data while variables are just one
